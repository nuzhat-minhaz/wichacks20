Virtual Reality for Learning & Awareness.

This product teaches the user a job skill, as well as the value of better product choices. It focuses on the quality of products and how they can be used in the real world, using both Organic and Biodynamic products by using the following VR components:

Oculus Rift VR: to project user into VR environment
Motion Leap (motion capture device): to track user's hands in real time.
